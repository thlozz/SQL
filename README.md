Projeto: Modelagem e Consultas SQL (ERP)
Repositório focado em demonstrar habilidades de banco de dados para sistemas de gestão (ERP).

🔍 O que este projeto demonstra:
Relacionamentos (Joins): Consultas que conectam tabelas de funcionários e departamentos.

Segurança e GRC: Uso rigoroso de filtros WHERE em atualizações e lógica de exclusão lógica (status_ativo) para preservar a integridade dos dados.

Agregação: Relatórios de custo e contagem de pessoal usando SUM e GROUP BY.
